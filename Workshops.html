Education

We believe a little push to start with can make wonders with our life. We have carefully designed Robotics workshops to make sure that students learn how things work on basic level and they can use that knowledge to explore the world of robotics...

Following are some standard workshops designed and classified in three levels depending upon student's age and knowledge level

DumpPro

DumpPro Robot is basically a manually controlled platform which uses Caterpillar type track belt locomotion. It is quite evident from the name that it will be used for dumping purpose, therefore it has a bucket which is lifted with the help of a gear box. The gear box is dominantly a mechanical assembly which uses a Worm gear assembly and hence can lift heavy loads. The Robotic platform is rear wheel driven wherein two high torque DC motors of 12V are used, a third motor is used in the gear box.

Beginner

SolarPro

The SolarPro photovore Robot utilizes very small high-efficiency motors in combination with elegant circuitry to make a very active solar powered robot capable of covering a meter (3.3 feet) distance in under a minute!

Beginner

MecPro

MecPro is the the by far best kit to learn basice mechanics and gear systems. Five Different Robots can be made using the same components

Beginner

PickPro

The PickPro robot is basically a manually controlled platform which uses Caterpillar type track belt locomotion. It is quite evident from the name that it will be used for pick & place operations, therefore it has a gripper with 1 DOF. The gear box is dominantly a mechanical assembly which uses a Worm gear assembly and hence can lift heavy loads. The platform is rear wheel driven wherein two high torque DC motors of 12V are used, additional two motors are used in the gear box and gripper assembly.

Intermediate

AutoPro

AutoPro is a Quickly Programmable Robot. As the name suggests this robot can be programmed very quickly and easily. Its also multipurpose robot for beginners, intermediates and experts.

It can easily be programmed to act as any of the following.

1.Line Follower Robot

2.Wall Hugger Robot

3.Object Tracer Robot

4.Light Tracer Robot

5.Hand Hugger Robot

6.Obstacle Avoider Robot

7.Intelligent Grid Mover Robot

8.Remote Controlled Robot

9.PC USB Controlled Robot

10.Maze Solver Robot

11.Sumo Robot

Intermediate

HexaPro

This HexaPro DIY kit focuses on development of a Hexapod using on 3 Servo motors. The robot is made from laser cut acrylic structural components . It is controlled through Rduino Atmega328 based USB programmable development board and has Dual TSOP based obstacle avider module for Autonomous mode. It can also be controlled via a wireless Infrared Remote.

Advanced

Humanoid

Bipedal Walking robot controlled by Rduino using 4 servo motors having 2 degree of freedom per leg. Fully assembled and tested kit with easy to use electronics with computer software.

Advanced
